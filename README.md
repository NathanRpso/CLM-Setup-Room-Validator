# CLM Setup — Room Validator

Interactive prototype for the Room Confirmation phase of the Rapsodo CLM PRO PC setup flow. Users input room dimensions, receive a compatibility result, and confirm a readiness checklist before proceeding to installation.

## Stack

React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
